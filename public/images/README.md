## Image Format Recommendations

- **Format**: PNG or WebP for best quality
- **Optimization**: Compress images before adding (use tools like TinyPNG or ImageOptim)
- **Size**: Keep file sizes under 500KB when possible
- **Resolution**: 2x resolution for Retina displays
